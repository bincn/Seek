## Seek

##### 一个新闻阅读 APP，还在不断完善中
##### seek 翻译为
- 寻找；探索

##### 大多数时候公司级项目不会第一时间使用上新的开源项目或者技术, 会有一些局限, 写这个项目的本意就是为了使用目前流行的开源项目和技术, 作为平时的研究积累。

##### 所以在这个项目中除了使用目前流行的 MVP 模式, Retrofit2 (OkHttp2), RxJava (RxAndroid), RxBus(替代 EvenBus), 还使用了其他一些主流的开源项目, 如：ButterKnife, Glide, leakcanary
